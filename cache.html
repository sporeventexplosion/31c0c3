Cache Replacement Policy Interactive Demo

More or less based on the implementations in the Rocket Chip Generator repository

The way that will be used on the next insert/replace is highlighted in yellow

Tags here simply show the order in which each way was inserted. In a real processor they would contain bits of the address

This interactive demo requires JavaScript to run
